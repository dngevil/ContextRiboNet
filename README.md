# ContextRiboNet

Deep learning framework for predicting translation efficiency (Ribo-seq) across human embryonic development stages using RNA sequences and cell-state contexts.

## Usage Guide
**For detailed implementation, data download links, and exact reproduction commands, please refer to the [readme.txt](code/readme.txt) file in this directory.**

## Requirements
- Python 3.8+ (PyTorch, Scikit-learn, Pandas)
- R 4.0+ (ggplot2, dplyr)
- MEME Suite (TOMTOM)
